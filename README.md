 Food Delivery Analysis

 Project Overview

This project analyzes food delivery data to uncover customer ordering patterns, restaurant performance, delivery efficiency, and business trends. Using Python for data cleaning and exploratory data analysis (EDA), and Power BI for interactive visualization, the project provides actionable insights to improve customer satisfaction and operational performance.

---

 Business Problem

Food delivery platforms generate large volumes of transactional data. Businesses need to understand customer behavior, delivery performance, and restaurant trends to improve service quality and maximize revenue.

This project answers questions such as:
- Which restaurants generate the highest revenue?
- What are the busiest ordering periods?
- How does delivery time affect customer satisfaction?
- Which locations receive the most orders?
- Which cuisines are most popular?

 Key Insights

- Identified the highest-performing restaurants based on revenue and order volume.
- Analyzed delivery times to identify operational bottlenecks.
- Determined the most popular cuisines and customer preferences.
- Compared payment methods and ordering trends.
- Built an interactive dashboard with dynamic filters for restaurant, cuisine, and date.

---

 Business Recommendations

- Optimize delivery routes to reduce delivery time.
- Promote high-performing restaurants during peak hours.
- Introduce targeted offers for popular cuisines.
- Improve service quality for restaurants with lower customer ratings.
- Allocate delivery resources based on demand trends.
